8. toList() - Convertir a List (Java 16+)

Forma simplificada de convertir a List inmutable.

// Forma moderna (Java 16+)
List lista = Stream.of("A", "B", "C")
    .toList();

// Forma tradicional (compatible con versiones anteriores)
List lista = Stream.of("A", "B", "C")
    .collect(Collectors.toList());
ðŸ’¡ Diferencia importante:

toList() devuelve una lista inmutable (no se puede modificar).

collect(Collectors.toList()) devuelve una lista mutable (se puede modificar).
