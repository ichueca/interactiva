4. anyMatch(), allMatch(), noneMatch() - Verificar condiciones

Verifican si alguno, todos o ninguno de los elementos cumplen una condici√≥n.

List numeros = Arrays.asList(1, 2, 3, 4, 5);

// ¬øAlguno es mayor que 3?
boolean algunoMayor = numeros.stream()
    .anyMatch(n -> n > 3);  // true

// ¬øTodos son positivos?
boolean todosPositivos = numeros.stream()
    .allMatch(n -> n > 0);  // true

// ¬øNinguno es negativo?
boolean ningunNegativo = numeros.stream()
    .noneMatch(n -> n < 0);  // true
