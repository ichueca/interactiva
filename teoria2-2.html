4. limit() - Limitar elementos

5. skip() - Saltar elementos

Omite los primeros N elementos del Stream.

Stream.of(1, 2, 3, 4, 5)

      // Resultado: 3, 4, 5

6. sorted() - Ordenar elementos

Ordena los elementos del Stream. Puede usar orden natural o un Comparator personalizado.

// Orden natural
Stream.of(3, 1, 4, 2)
      .sorted()
      // Resultado: 1, 2, 3, 4

// Con Comparator
Stream.of("Ana", "Pedro", "Luis")
      .sorted(Comparator.reverseOrder())
      // Resultado: "Pedro", "Luis", "Ana"
